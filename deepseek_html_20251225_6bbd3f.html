Как установить?

1

Установите Legacy Launcher

Скачайте и установите Legacy Launcher с официального сайта.

2

Установите Java 17

Установите Java 17 версии для корректной работы клиента.

3

Запустите версию 1.16.5

В Legacy Launcher запустите версию Minecraft 1.16.5 для создания необходимых файлов.

4

Замените файл

Удалите из папки версии файл 1.16.5 и замените его на скаченный файл (надо переименовать в 1.16.5).
